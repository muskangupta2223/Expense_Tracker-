# Expense Tracker

A full-stack web application to help users easily track, categorize, and visualize their expenses.

## Project Overview  
With this application, users can sign up / log in, add and manage expense entries, create custom categories, view summaries via charts, import expenses from CSV, and maintain a profile. The backend API is built using Node.js + Express, and the frontend uses Angular for a responsive, interactive experience.

##  Features  
- User authentication & authorization (sign-up, login, secure access)  
- Add, edit, delete expense entries  
- Add custom categories for expenses  
- View expenses list + summary dashboards (bar chart, pie chart)  
- Import expenses from CSV file for quick bulk entry  
- Profile management (view / update name, username)  
- Auto logout for inactive sessions (for added security)  
- Responsive UI for use on desktop & mobile
